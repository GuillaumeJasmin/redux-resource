### Importing

Every function or object described above is a top-level export. You can import
any of them like this:

#### ES6

```js
import { resourceReducer } from 'redux-resource';
```

#### ES5 (CommonJS)

```js
var resourceReducer = require('redux-resource').resourceReducer;
```
